Quick Start Guide

Size

The size can be specified using the following formats

https://imgfaker.onrender.com/600x400 Copy
https://imgfaker.onrender.com/600/400 Copy

Also height is optional, useful for creating a square:

https://imgfaker.onrender.com/600 Copy

Colour

Specify background and text colours using hex values

https://imgfaker.onrender.com/600/400/eee/999 Copy

using CSS names

https://imgfaker.onrender.com/600/400/green/white Copy

Also you can combine both

https://imgfaker.onrender.com/600/400/orange/fff Copy

Text

Specify text using query string

https://imgfaker.onrender.com/600/400?text=imageFaker Copy

Font

Specify font using query string

https://imgfaker.onrender.com/600/400?font=lora Copy

The following fonts are available

Default is Roboto
